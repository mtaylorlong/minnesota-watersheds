Sources

We built this site and mapping application using the following resources and data sources.

MN Watershed Data

MN DNR Data Deli: http://deli.dnr.state.mn.us

  • Major watersheds, minor watersheds, Level 08 catchments, and invasive species reports were all obtained from the Minnesota Department of Natural Resources.

US Watershed Data

USGS: http://water.usgs.gov/maps.html

  • United States watersheds, rivers, and digital elevation model used in creating maps on the Watersheds page were obtained from the USGS Water Resources Maps and GIS Data page.

Other Watershed Page Sources:

  • Seaber, P. R. (1987). In Kapinos F. P., Knapp G. L. and Geological Survey (U.S.) (Eds.), Hydrologic unit maps. Dept. of the Interior, U.S. Geological Survey ; Denver, CO.

MN Population Data

NHGIS: https://www.nhgis.org/

  • Minnesota Population Center. National Historical Geographic Information System: Version 2.0. Minneapolis, MN: University of Minnesota 2011.

Development Platforms

ArcGIS API for Javascript: https://developers.arcgis.com/en/javascript/

  • The Explore map was built using this javascript API.

Python: http://www.python.org/

  • The Upstream/Downstream script was written in Python.

ArcServer: http://www.esri.com/software/arcgis/arcgisserver

  • The Upstream/Downstream script was hosted as a service on ArcServer.

ArcGIS Viewer for FLEX: http://resources.arcgis.com/en/communities/flex-viewer/

  • The Upstream/Downstream tool was incorporated into a website using FLEX.

HTML/CSS: http://htmlandcssbook.com

  • We coded the site with our own HTML and CSS.

Github: https://github.com/

  • We used Github to develop and host the site's code. Check out the site repository at https://github.com/mtaylorlong/mtaylorlong.github.io/tree/master/Watersheds.
